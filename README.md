# CyberIncident_Project
Cyber Incident Alert & Response System
A full-stack cybersecurity project designed to detect, manage, and respond to suspicious cyber activities in real time. The system helps security teams monitor incidents, generate alerts, track attack details, and manage responses through an interactive dashboard.

**#Features**
Real-time cyber incident monitoring
Incident alert generation and tracking
Email notification system for critical threats
User authentication and role-based access
Dashboard for incident management
Incident status updates (Open, Investigating, Resolved)
Secure database integration
Search and filter incident reports
Responsive and user-friendly interface
**Technologies Used**
Frontend: HTML, CSS, JavaScript, Bootstrap
Backend: Java, JSP, Servlets
Database: MySQL
Server: Apache Tomcat
Tools: JDBC, Git, GitHub
